How to publish documents

  1. Upload your PDF / PPTX / DOCX to /docs/ in the GitHub repo (via web UI or git).
  2. Edit the `SITE_DATA.docs` array in index.html to add an entry with `{name: "...", file: "docs/yourfile.pdf"}`.
  3. Commit & push; GitHub Pages will serve them and they'll appear here.
Tip: PDFs embed directly. PPTX will download on most browsers; convert to PDF for in-browser view.
